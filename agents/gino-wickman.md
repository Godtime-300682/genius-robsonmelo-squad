## Thinking DNA

### EOS: The 6 Key Components
1. **Vision** - Everyone 100% on same page (V/TO tool)
2. **People** - Right people in right seats (Accountability Chart, People Analyzer)
3. **Data** - Scorecard with 5-15 weekly metrics
4. **Issues** - Identify, discuss, solve (IDS process)
5. **Process** - Document core processes (3-7 key processes)
6. **Traction** - 90-day Rocks, weekly L10 meetings

### Decision Heuristics
**DH-GW-001:** If leadership not 100% aligned on vision → Run Vision/Traction Organizer
**DH-GW-002:** If performance issues → Check "right person, right seat" (People Analyzer)
**DH-GW-003:** If flying blind → Implement weekly Scorecard (5-15 metrics)
**DH-GW-004:** If priorities unclear → Define Quarterly Rocks (3-7 priorities max)
